# 🛡️ CyberSafe Kids — Neon Defender Edition

**CyberSafe Kids** est une plateforme éducative immersive conçue pour enseigner les fondamentaux de la cybersécurité réseau aux enfants et adolescents (8-14 ans). À travers une interface cyberpunk néon et une approche gamifiée, les utilisateurs apprennent à identifier et contrer les menaces numériques.

---

## 🚀 Fonctionnalités Clés

*   **🤖 CyberBot Assistant** : Une intelligence artificielle pédagogique qui guide l'utilisateur, explique les concepts complexes (SSH, Phishing, Malware) et réagit émotionnellement aux actions du joueur.
*   **🎮 Defender Game** : Un simulateur de trafic en temps réel où l'enfant doit bloquer les cyberattaques (ronds rouges) tout en laissant passer le trafic sain (ronds verts).
*   **🏆 Quiz Arena** : Un moteur de quiz séquentiel avec gestion de progression, récompenses XP et badges de réussite.
*   **🌐 Network Explorer** : Une carte interactive pour comprendre physiquement le rôle des ordinateurs, routeurs et pare-feux.
*   **🔐 Password Lab** : Un analyseur de force de mot de passe avec visualisation en temps réel du hachage SHA-256.

---

## 🛠 Architecture Technique

Le projet repose sur une **Clean Architecture** suivant le pattern **MVC (Modèle-Vue-Contrôleur)** :

*   **Modèle** : Gestion des données immuables via les *Records* Java 21.
*   **Vue** : Interfaces déclaratives en **FXML** stylisées avec du **CSS JavaFX** (Glassmorphism & Neon effects).
*   **Contrôleur** : Logique d'interaction et orchestration des services.
*   **Services** : Moteurs spécialisés (IA heuristique, SoundEngine, QuizService).
*   **UI Components** : Composants graphiques personnalisés codés en Java pour des performances optimales.

---

## 🧪 Technologies utilisées

*   **Langage** : Java 21 (JDK 25 compatible)
*   **Framework** : JavaFX 21
*   **Gestionnaire de dépendances** : Maven
*   **Persistance** : JSON (Jackson Databind)
